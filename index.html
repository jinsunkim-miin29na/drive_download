다운로드 개수 비교

Drive 확인 후 성공 개수를 입력하면 차이를 계산합니다.

단축어가 해야 할 일

  1. `Drive Album Save` 단축어가 입력 JSON의 `files` 목록을 반복합니다.
  2. 각 파일의 `downloadUrl`을 가져와 사진 앱에 저장하고 `albumName` 앨범에 추가합니다.
  3. 성공한 파일 수를 세고, 마지막에 웹페이지 콜백 주소를 열면 자동 비교됩니다.

공유 폴더가 비공개이거나 회사 계정 로그인으로만 보이면 API key만으로는 목록 확인이 안 됩니다. 그 경우에는 Google OAuth 또는 별도 서버가 필요합니다.
